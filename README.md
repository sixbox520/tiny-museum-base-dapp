# Tiny Museum

Una app pequeña en Base para tiny exhibits. La experiencia no intenta ser enorme: conectar wallet, curating a mini object, y guardar un exhibit label.

## Desarrollo local

```bash
npm install
npm run dev
```

Hecho con TypeScript frontend, Base wallet flow, public proof metadata.

Regla básica: Do not commit `.env`, private keys, seed phrases, RPC keys, GitHub tokens, or Vercel tokens. Use `.env.example` only for placeholders.
